# Spam Detection System Using Machine Learning 

## Overview 
The Spam Detection System is a Machine Learning and Natural Language Processing(NLP) project designed to classify
messages as "Spam" or "Ham (Not Spam)". The Application analyzes the content of a message and predicts
whether it is legitimate or unwanted using a trained machine learning model.
This project demonstrates the practical application of Machine Learning, Text Processing,
and Web Development in solving real world commumication security problems.

## Live Demo
 Live Application: 
 The  Application is deployed online and can be accessed directly through a web browser
 without requiring any local setup.

## How it works
 1. Load the spam dataset
 2. clean and preprocesses the text
 3. convert text into numerical features using TF-IDF
 4. Train a Machine Learning classifier
 5. save the trained model
 6. predict whether a new message is spam or Ham
 7. Display results through a Flask web application
